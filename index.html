⭐ Live Sync — Google Sheet (Recommended)

Ek baar Google Apps Script deploy karein, phir sirf ☁ Sync Sheet button dabayein — koi manual download/upload nahi.

Step 1: Apni Google Sheet kholein → Extensions → Apps Script
Step 2: Neeche diya hua code paste karein aur Deploy → New deployment → Web app karein
   — Execute as: Me  |  Who has access: Anyone
Step 3: Deploy hone ke baad jo URL mile, usse neeche paste karein

📋 Google Apps Script Code — Copy & Paste
function doGet(e) {
  var ss = SpreadsheetApp.openById('1YYZ9kRbrT8HYXjA-neTnLIq1BgV3uXr3p9M9KhvN3uM');
  var sheet = ss.getSheets().find(function(s){
    return s.getName().toLowerCase().indexOf('dispatch') !== -1;
  });
  if (!sheet) return ContentService.createTextOutput(JSON.stringify({error:'Dispatch sheet not found'})).setMimeType(ContentService.MimeType.JSON);

  var data = sheet.getDataRange().getValues();
  var result = JSON.stringify(data);
  return ContentService.createTextOutput(result)
    .setMimeType(ContentService.MimeType.JSON);
}
